# Descripción

Este proyecto implementa un sistema académico utilizando Árboles Binarios de Búsqueda (ABB) en Java.

El sistema permite registrar estudiantes y realizar operaciones como:

- Insertar estudiantes
- Buscar estudiantes
- Eliminar estudiantes
- Recorridos del árbol
- Contar nodos
- Calcular altura
- Buscar mayor y menor nota
- Mostrar aprobados y reprobados

---
